# 外壳组装
面板下单注意事项：
- 面板类型：亚克力面板
- 面板款数：多款自主拼版
- 面板尺寸：192.78mm 95mm
- 拼版套数：1套
- 不需要确认生产稿
- 透明亚克力，厚度1.5mm或2mm
- 打印方式：正面
- 遮光强度：常规
- 不需要背胶

拿到面板后，要用螺丝等紧固件装配，需要准备的材料如下表所示
|        零件        | 数量  |
| :----------------: | :---: |
|  M2*5 扁平头螺丝   |   8   |
|  M2*8双通六角铜柱  |   4   |
| M2*8+3单通六角铜柱 |   8   |
|      M2螺丝帽      |   8   |
